DFWR PDF Generator

Proof of concept — generates the PDF directly with pdfmake (no browser print, no Safari headers).

1. Import records

Use one of your existing DFWR export files (the same JSON the current tool saves). Every record in it appears here.

2. Company info (for the footer)

These are only used to render the PDF footer/header in this test. Nothing is saved.

3. Generate

On desktop, "Generate PDF" downloads the file. On iPad, use "Open in new tab", then tap the Share icon and choose Save to Files — this is the save flow we're validating.
